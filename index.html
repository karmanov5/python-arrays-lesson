🗄️ Что такое массив?

Представь коробку с ячейками 📦

У тебя есть коробка с отдельными ячейками. В каждой ячейке лежит одно число. Коробка называется массив, а ячейка — элемент.

значения ↓
70
121
52
993
34
↑ индексы

В Python массив называется список (list). Он может хранить числа, слова — всё что угодно. Но в задачах по информатике чаще всего хранят числа.

💡 Главное правило: нумерация ячеек начинается с нуля, а не с единицы!

🔢 Индексы — это номера ячеек

Каждая ячейка массива имеет свой номер — индекс. Именно по этому номеру мы обращаемся к нужной ячейке.

7A[0]
12A[1]
5A[2]
99A[3]
3A[4]
# Массив из 5 элементов
A = [7, 12, 5, 99, 3]

# Обращение к элементам
print(A[0])  # → 7  (первый!)
print(A[3])  # → 99
print(A[4])  # → 3  (последний)
⚠️ Частая ошибка: думать, что первый элемент — это A[1]. Нет! Первый — A[0], второй — A[1], и так далее.

🛠️ Как создать массив?

Есть несколько способов. Выбери тот, который нужен в задаче.

1

Написать числа вручную

A = [7, 12, 5, 99, 3]

Все числа сразу. Используй, когда знаешь все элементы заранее.

2

Пустой массив + добавление

A = []
A.append(7)   # добавить 7
A.append(12)  # добавить 12

append() — добавляет новый элемент в конец массива.

3

Заполнить нулями

A = [0] * 5  # [0, 0, 0, 0, 0]

Создаёт массив из 5 нулей. Удобно, когда нужны «пустые» ячейки.

4

Случайные числа

from random import randint
A = []
for i in range(5):
    A.append(randint(1, 100))

Генерирует 5 случайных чисел от 1 до 100. Часто используется в задачах и тестах.

📏 Размер массива: len()

Функция len(A) возвращает количество элементов в массиве. Это очень важная функция!

70
121
52
993
34
A = [7, 12, 5, 99, 3]
n = len(A)
print(n)  # → 5
💡 Индекс последнего элемента всегда = len(A) - 1.
Если в массиве 5 элементов → последний это A[4]

🎯 Зачем это нужно?

Когда ты пишешь программу, ты часто не знаешь заранее, сколько элементов в массиве. len() даёт тебе это число, и ты можешь использовать его в цикле. 

# Вывести последний элемент
print(A[len(A) - 1])  # → 3

# Или короче:
print(A[-1])  # → 3 (с конца!)

🔄 Перебор массива: цикл for

Чтобы обработать каждый элемент массива — нужно пройти по всем ячейкам по очереди. Для этого используют цикл for.

A = [7, 12, 5, 99, 3]

for i in range(len(A)):
    print(A[i])

🗺️ Как это работает?

range(len(A)) создаёт числа от 0 до 4 (то есть: 0, 1, 2, 3, 4).
На каждом шаге i равно очередному числу.
A[i] — значение элемента с этим номером.

➕ Сумма элементов

Нам нужно сложить все числа в массиве. Как это сделать программно?

🧮 Способ 1: Вручную (через цикл)

Заводим переменную-копилку s = 0 и добавляем к ней каждый элемент по очереди.

A = [7, 12, 5, 99, 3]

s = 0  # копилка для суммы
for i in range(len(A)):
    s = s + A[i]

print(s)  # → 126

Как меняется s на каждом шаге:

i=0: s = 0 + 7 = 7
i=1: s = 7 + 12 = 19
i=2: s = 19 + 5 = 24
i=3: s = 24 + 99 = 123
i=4: s = 123 + 3 = 126

⚡ Способ 2: Встроенная функция

s = sum(A)
print(s)  # → 126

Python умеет сам считать сумму — функция sum(). Но в задачах по информатике часто просят написать алгоритм вручную.

🔢 Подсчёт элементов

Иногда нужно посчитать, сколько элементов в массиве удовлетворяет какому-то условию. Например, сколько чисел больше 10.

🧮 Алгоритм подсчёта

Заводим переменную-счётчик k = 0. На каждом шаге проверяем условие. Если оно выполнено — добавляем 1 к счётчику.

A = [7, 12, 5, 99, 3]

k = 0  # счётчик
for i in range(len(A)):
    if A[i] > 10:
        k = k + 1

print(k)  # → 2  (это 12 и 99)

Пошагово:

A[0]=7 7 > 10? Нет. k = 0
A[1]=12 12 > 10? Да! k = 1
A[2]=5 5 > 10? Нет. k = 1
A[3]=99 99 > 10? Да! k = 2
A[4]=3 3 > 10? Нет. k = 2
💡 k = k + 1 можно записать короче: k += 1 — они означают одно и то же.

🏆 Поиск максимума и минимума

Как найти наибольшее или наименьшее число в массиве?

🏅 Алгоритм поиска максимума

Представь, что ты устраиваешь соревнование. Берёшь первого участника и говоришь: «Пока ты — чемпион». Затем по очереди сравниваешь его с каждым следующим. Нашёл кого-то сильнее — новый чемпион!

A = [7, 12, 5, 99, 3]

maximum = A[0]  # первый - чемпион
for i in range(1, len(A)):
    if A[i] > maximum:
        maximum = A[i]

print(maximum)  # → 99

print(max(A))  # → 99
print(min(A))  # → 3

🪄 Суперсилы Python

Python уже умеет делать всё сам! Вот встроенные функции для работы с массивами:

sum(A)

sum([7,12,5,99,3]) → 126
min(A)
Минимальный элемент
min([7,12,5,99,3]) → 3
max(A)
Максимальный элемент
max([7,12,5,99,3]) → 99
sorted(A)
Сортировка (новый список)
sorted([7,12,5,99,3]) → [3,5,7,12,99]
A.sort()
Сортировка на месте (меняет A)
A.sort() # [3,5,7,12,99]
len(A)
Количество элементов
len([7,12,5,99,3]) → 5
